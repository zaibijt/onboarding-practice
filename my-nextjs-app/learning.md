# My Next.js Learning Project

## Components Created
1. **Header** - Displays the app title
2. **Card** - Shows a title and description
3. **Button** - Clickable button with alert functionality

## Pages
- **Home Page** (`/`) - Displays all components
- **About Page** (`/about`) - Simple info about me

## API
- **Hello API** (`/api/hello`) - Returns JSON message: {"message": "Hello from my API!"}

## Learning Notes
- Learned how to use **Next.js App Router** for pages
- Created **client-side components** with `'use client'`
- Styled components using **Tailwind CSS**
- Learned to structure a Next.js project for Git and feature branch workflow
